17 /**
18  * Analog input class.
19  *
20  * Connected to each analog channel is an averaging and oversampling engine.
21  * This engine accumulates
22  * the specified ( by SetAverageBits() and SetOversampleBits() ) number of
23  * samples before returning a new
24  * value. This is not a sliding window average. The only difference between
25  * the oversampled samples and
26  * the averaged samples is that the oversampled samples are simply accumulated
27  * effectively increasing the
28  * resolution, while the averaged samples are divided by the number of samples
29  * to retain the resolution,
30  * but get more stable values.
31  */
