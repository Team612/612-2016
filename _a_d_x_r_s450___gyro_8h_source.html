17 /**
18  * Use a rate gyro to return the robots heading relative to a starting position.
19  * The Gyro class tracks the robots heading based on the starting position. As
20  * the robot rotates the new heading is computed by integrating the rate of
21  * rotation returned by the sensor. When the class is instantiated, it does a
22  * short calibration routine where it samples the gyro while at rest to
23  * determine the default offset. This is subtracted from each sample to
24  * determine the heading.
25  *
26  * This class is for the digital ADXRS450 gyro sensor that connects via SPI.
27  */
