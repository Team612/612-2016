14 /**
15  * Use a rate gyro to return the robots heading relative to a starting position.
16  * The Gyro class tracks the robots heading based on the starting position. As
17  * the robot
18  * rotates the new heading is computed by integrating the rate of rotation
19  * returned
20  * by the sensor. When the class is instantiated, it does a short calibration
21  * routine
22  * where it samples the gyro while at rest to determine the default offset. This
23  * is
24  * subtracted from each sample to determine the heading. This gyro class must be
25  * used
26  * with a channel that is assigned one of the Analog accumulators from the FPGA.
27  * See
28  * AnalogInput for the current accumulator assignments.
29  *
30  * This class is for gyro sensors that connect to an analog input.
31  */
